Agent Mind as a Service API documentation version 0.1

http://localhost:8080/

Welcome to the documentation for the agent mind as a service API, the service hosts agents minds for a model that uses mixed indirect and direct reciprocity mechanisms. These agents have their own memory of events (input as percepts) which the agent then uses to form beliefs about their world upon which they can act.

/agent

The individuals in the system that perceive events, can believe things about their world and act on them. They are part of a generation in a community.

/agent post

post

Create a new agent the belongs to an existing community and generation in the service.

post /agent

Create a new agent the belongs to an existing community and generation in the service.

  • Request
  • Response

Query Parameters

  • strategy: required (string)

    The strategy for the new agent to have

  • options: required (array)

    The options for the strategy that the new agent will have

  • community: required (integer)

    The id for the pre-existing community the agent belongs to

  • generation: required (integer)

    The id for the pre-existing generation the agent belongs to

  • player: required (integer)

    The id for the new agent

HTTP status code 200

Body

Media type: application/json

Type: any

{

/percept

Agents perceive events and happenings in an environment, input percepts to them.

/percept/action

A particular type of percept where an action has been taken and the agent has perceived it.

/percept/action/interaction post

An interaction between a donor-recipient pair where the donor has committed to an action.

post

Give an agent a percept of an interaction between a donor-recipient pair of agents at a specific timepoint and the action (defect or cooperate) that the donor took.

post /percept/action/interaction

Give an agent a percept of an interaction between a donor-recipient pair of agents at a specific timepoint and the action (defect or cooperate) that the donor took.

  • Request
  • Response

Query Parameters

  • community: required (integer)

    The id for the community that the perceiver belongs to

  • generation: required (integer)

    The id for the generation that the perceiver belongs to

  • perceiver: required (integer)

    The id for the perceiver

  • action: required (string)

    The action (defect or cooperate) that the perceiver is perceiving

  • donor: required (integer)

    The agent who has commited the action

  • recipient: required (integer)

    The agent who is on the receiving end of the action

  • timepoint: required (integer)

    The timepoint at which this interaction took place

HTTP status code 200

Body

Media type: application/json

Type: any

Example:

{
  "data": {
    "community": 45,
    "generation": 3,
    "perceiver": 1,
    "action": "defect",
    "donor": 3,
    "recipient": 4,
    "timepoint": 3
  },
  "success": true,
  "status": 200
}

/action

An action is taken by an agent at a specific timepoint, this can either be idle, gossip or action.

/action get

get

Get the action that an agent would choose to do at a particular timepoint with the beliefs that have been built from perceptions. The action can either be idle, gossip (positive or negative, with about and recipient parameters) or action (cooperate or defect, with recipient parameters)

get /action

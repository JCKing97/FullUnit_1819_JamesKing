Get an agents commitment to an action, responds unsuccessful if there is no such community, generation of the community or agent belonging to the generation of the community passed.
Arguments:
Timepoint- The timepoint at which to get the action commitment at
CommunityID- The community the agent belongs to
GenerationID- The generation of the community the agent belongs to
AgentID- The id of the agent
Success- An output argument whether getting the action was successful (becomes true) or not (becomes error message)
Action- An output argument which is a dictionary representing the action commitment made

 get_donor_belief(++CommunityID:int, ++GenerationID:int, ++AgentID:int, ++Timepoint:int, -Success:atom, -Value:list) is nondet
Get an agents beliefs on all the times that they have been a donor before the given timepoint
Arguments:
Timepoint- The timepoint at which to check for donor beliefs before
CommunityID- The community the agent belongs to
GenerationID- The generation of the community the agent belongs to
AgentID- The id of the agent
Success- An output argument whether getting the belief was successful (becomes true) or not (becomes error message)
Value- An output list of the times this agent has been a donor, empty if the agent hasn't

 get_standing_belief(++CommunityID:int, ++GenerationID:int, ++Timepoint:int, ++PerceiverID:int, ++AboutID:int, -Success:atom, -Value:list) is nondet
Get the agent with the id PerceiverID's belief on the standing of the player with id AboutID (either good or bad), Is unsuccessful if PerceiverID is not using the standing strategy, or the specified community, generation or agents are incorrect.
Arguments:
Timepoint- The timepoint at which to check for standing beliefs before
CommunityID- The community the agents belong to
GenerationID- The generation of the community the agents belong to
PerceiverID- The id of the agent who holds the belief about the other
AboutID- The id of the agent who the belief holds about
Success- An output argument whether getting the belief was successful (becomes true) or not (becomes error message)
Value- An output argument which is either good or bad under normal circumstances, or false under error circumstances

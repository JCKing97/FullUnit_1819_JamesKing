 initiates_at(++Event:term, @Options:list, ?Fluent:term, --T:int) is nondet[multifile]
Initiate a certain value for a fluent at the given timepoint T, based on the event passed in Event.
Arguments:
Event- The event that may have happened
Options- A list of options (keep empty for now)
Fluent- The fluent and value that will now hold if the event happened and logic is satisfies
T- The timepoint at which this fluent was initiated
 trusted(++Perceiver:term, ++Subject:term, ++T:int) is det
true if the perceiver is a veritability discerner and trusts the subject at timepoint T, else fails. an agent is trusted if their veritability_rating/percept_count >= K (set into the agents strategy)
Arguments:
Perceiver- The agent that has beliefs about the Subject
Subject- The agent that the Perceiver trusts or does not trust
T- The timepoint at which the Perceiver either does or does not trust the Subject

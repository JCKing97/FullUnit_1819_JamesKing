 initiates_at(++Event:term, @Options:list, ?Fluent:term, --T:int) is nondet[multifile]
Initiate a certain value for a fluent at the given timepoint T, based on the event passed in Event.
Arguments:
Event- The event that may have happened
Options- A list of options (keep empty for now)
Fluent- The fluent and value that will now hold if the event happened and logic is satisfies
T- The timepoint at which this fluent was initiated
